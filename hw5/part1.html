A form that contains:

  • An input text box for a full name
  • A group of radio buttons so for age group (less than 21; between 21 and 50; older than 50)
  • A group of checkboxes for browsers used (Firefox, Edge, Chrome, Safari)
  • A select element or datalist (to produce a drop-down list of choices) for genre of movie preferred (Action, Comedy, Drama, Documentary, Science Fiction)

The page will validate whether the user has entered data in the input text box, AND has checked one of the radio buttons, AND has checked at least one checkbox, AND has selected an option from the list of options in the select (or datalist) element.
