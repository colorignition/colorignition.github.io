About Progressive Web Applications (PWAs)

A Progressive Web Application (PWA) is a type of web application that is installed on a device, or more specifically, the offline cache of a web browser. PWAs are different from native apps in that they are built using coding language for browsers (HTML, CSS, JavaScript, etc.), and therefore are able to run on any device with a web browser, unlike native apps which are built specifically for their target platforms and are not easily ported. PWAs are also more advantageous to web-only apps because of its integration into web browsers, allowing for more richer experiences.

PWAs are generally reliable, fast, and engaging. PWAs are reliable because they are able to load itself with common resources that have been cached offline, reducing reliance on network connection. PWAs are also fast because of the same reason; there is much less waiting time because the app can quickly load most assets from local storage and therefore the amount of loading from the network is kept to a minimum. Finally, the PWAs are engaging because they offer a similar experience to a native app, with many features and access to web push notifications.
