A web page that will save TWO pairs of cookies for the same page - one of the pairs for the name of the user and another one for the username of the user. Both cookies are deleted when the user ends the seesion by closing the browser (not simply closes the tab/window).

If it is the first time the user is opening the page on the browser, the page will present a form for the user to type his/her name and his/her username.

If it is not the first time the user is opening the page (if he/she is revisiting the page) while the browser is still opened, the page will just display a message of Welcome Back followed by the name of the user that the page retrieves from the previously saved cookie.
